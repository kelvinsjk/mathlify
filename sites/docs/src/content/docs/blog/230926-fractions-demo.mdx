In this post, we explore the use of Mathlify to typeset questions
and answers involving fractions. We hope that you will find
that Mathlify helps you author math content with
idiomatic code that is easy to write, read and maintain.

## OpenStax Elementary Algebra 2e

The questions are taken from the review exercises at the end of chapter 1 in
[OpenStax Elementary Algebra 2e](https://openstax.org/books/elementary-algebra-2e/pages/1-review-exercises),
under the [Creative Commons Attribution License](https://creativecommons.org/licenses/by/4.0/).

Access for free at https://openstax.org/books/elementary-algebra-2e/pages/1-introduction

## Simplify Fractions

import { Fraction } from "mathlify";
import katex from "katex";
export const q1070 = function () {
  const num = 8;
  const den = 24;
  const f = new Fraction(num, den);
  const soln = `\\frac{${num}}{${den}} = ${f}`;
  return {
    toString: function () {
      return katex.renderToString(soln);
    },
  };
};

> **Qn 1070.**
> {q1070()}

### Code sample

```js
const num = 8;
const den = 24;
const f = new Fraction(num, den);
const soln = `\\frac{${num}}{${den}} = ${f}`;
// \frac{8}{24} = \frac{1}{3}
```

## Use the Order of Operations to Simplify Complex Fractions

export const q1140 = function () {
  const num = new Fraction(1, 3);
  const d1 = new Fraction(5);
  const d2 = new Fraction(2);
  const soln = `\\displaystyle \\frac{\\left(${num}\\right)^2}{${d1} + ${d2}^2} = ${num
    .square()
    .divide(d1.plus(d2.square()))}`;
  return {
    toString: function () {
      return katex.renderToString(soln);
    },
  };
};

> **Qn 1140.**
> {q1140()}

### Code sample

```js
const num = new Fraction(1, 3);
const d1 = new Fraction(5);
const d2 = new Fraction(2);
const soln = `\\frac{\\left(${num}\\right)^2}{${d1} + ${d2}^2}
	= ${num.square().divide(d1.plus(d2.square()))}`;
// \frac{\left(\frac{1}{3}\right)^2}{5 + 2^2} = \frac{1}{45}
```
