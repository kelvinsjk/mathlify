In this post, we explore the use of Mathlify to typeset questions
and answers involving terms. We hope that you will find
that Mathlify helps you author math content with
idiomatic code that is easy to write, read and maintain.

## The `Term` class

In Mathlify, the `Term` class is used to represent elements under multiplication. Formally,
it handles expressions of the form
$k \displaystyle \prod_i^N x_i^{n_i}$,
where $k \in \mathbb{Q}$, $n_i \in \mathbb{Q}\setminus\{0\}$, and the variables
$x_i$ are represented by strings internally.

The `Term` constructor handles automatic simplification by grouping like terms together under exponentiation.

## Evaluate Variable Expressions with Fractions

import { Fraction, Term } from "mathlify";
import katex from "katex";

export const q1146 = function () {
  const term = new Term(5, {variable:'m', power:2}, 'n');
  // shorthand: new Term(5, ['m', 2], 'n');
  const m = new Fraction(-1,2);
  const n = new Fraction(1,3);
  const nString = `n=${n}`;
  const qn = `Evaluate the expression 
    ${katex.renderToString(`${term}`)}
    when ${katex.renderToString(`m=${m}`)}
    and ${katex.renderToString(`n=${n}.`)}
  `;
  const ans = katex.renderToString(`${term.subIn({m,n})}`);
  return {
    toString: function () {
      return `<p><strong>Question:</strong> ${qn}</p>
        <p><strong>Answer:</strong> ${ans}</p>`;
    },
  };
};

> **Chapter 1 Qn 1146.**
> {q1146()}

### Code sample

```js
const term = new Term(5, {variable:'m', power:2}, 'n');
// shorthand: new Term(5, ['m', 2], 'n');
const m = new Fraction(-1,2);
const n = new Fraction(1,3);
// uses the math function from the mathlifier library
const qn = `Evaluate the expression
  ${math(term)}
  when ${math(`m=${m}`)}
  and ${math(`n=${n}.`)}
`;
const ans = term.subIn({m,n});
// outputs \frac{5}{12} when converted to string
```

## Simplify Expressions by Applying Several Properties

export const q706 = function () {
  const v3 = new Term({variable: 'v', power: 3});
  // using the shorthand notation
  const v9 = new Term(['v', 9]);
  const v6 = new Term(['v', 6]);
  const n = 4;
  const qn = `\\left( \\frac{${v3}${v9}}{${v6}} \\right)^{${n}}`;
  // using chaining to simplify the expression to get the answer
  const ans = v3.times(v9).divide(v6).pow(4);
  return {
    toString: function () {
      return katex.renderToString(`${qn} = ${ans}`);
    },
  };
};

> **Chapter 6 Qn 706.**
> {q706()}
